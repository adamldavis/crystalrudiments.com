A game about exploring the red planet and science. Be the first to explore Mars and gain the most scientific data playing as one of four teams: USA, ESA, China, or Final Frontier (a private space company) using drones, mining, solar panels, and many other elements.
The object of the game is to get the most Science. Players keep track of E-credit (Ⓔ) using paper money, scientists and engineers using game pieces, and Science (S) points using physical tokens, all included. Cards become available and can be activated as the game progresses. Chance cards add an element of luck and fun to the game.
This Card Edition saves space by using five jumbo-sized cards to replace the game board, but otherwise is basically the same as the full version.
Expand game play by creating your own cards with included blank cards, decreasing starting credits, and adding several other rule enhances to the game.
